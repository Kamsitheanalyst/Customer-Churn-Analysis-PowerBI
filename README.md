# Customer Churn Analysis Dashboard (Power BI)

## Project Overview
This project analyzes customer churn data to identify key drivers of churn and provide actionable recommendations to improve customer retention.

## Tools Used
- Power BI (DAX Measures, Dashboarding)
- MySQL (Data storage and preparation)
- Excel (Dataset source)

## Dataset Description
The dataset contains customer information such as age, gender, tenure, usage frequency, support calls, payment delays, subscription type, contract length, total spend, last interaction, and churn status.

## Key Business Questions
- What is the overall churn rate?
- Which subscription type has the highest churn?
- Which contract length has the highest churn?
- How do support calls influence churn?
- How does payment delay affect churn?

## Key Insights
- Churn rate is extremely high, indicating major retention challenges.
- Subscription type impacts churn, with Premium customers showing high churn volume.
- Support calls and payment delays are strong churn indicators.
- Contract length influences churn patterns.

## Recommendations
- Improve customer support response time and service quality.
- Introduce loyalty programs and retention incentives.
- Offer targeted discounts for high-risk customer groups.
- Improve payment flexibility and reminders.
